## Building

If you wish to build this project you will need [Apache Maven](https://maven.apache.org/) installed.

Run these commands to fetch the source and build:

1. `git clone https://github.com/bordertech/java-sanitizer.git`
2. `cd java-sanitizer`
3. `mvn install`
